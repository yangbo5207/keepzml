    专栏简介

    CSS 薄弱是许多前端的通病。

    许多前端开发由于工作的原因被限在后台管理系统的泥潭里，他可能写功能没有问题，但是让他写一个页面的布局，那问题就层出不穷了。

    而且 CSS 发展了这么多年，内容也越来越多，想要把它掌握好也并不是一件容易的事情。

    《超级 CSS》基于这样的现状，结合作者多年来的实战开发经验，专门针对 C 端应用通过实战案例演示的方式，为大家完整的阐述高级前端开发的 CSS 需要掌握到什么程度。
